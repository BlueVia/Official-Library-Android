Detailed Description

Class representing the Sms Message request that will be send using the SMS Client API This type is composed of the following fields:

  • addressList: mandatory; max occurrences: minimum 1.
  • message: mandatory; String
  • originAddress: mandatory; String

This implementation is not synchronized
