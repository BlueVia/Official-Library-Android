Detailed Description

Abstract representation of an HTTP REST Client for the Bluevia API

Available clients may be

  • SMS Client API
  • MMS Client API
  • Directory Client API
  • Advertisement Client API
  • Oauth Client API
  • Location Client API
  • Payment Client API

Member Enumeration Documentation

enum Mode

The possible values to determinate the instantiated connector for the client.

Available values may be

  • HTTP_LIVE: Connects with the endpoint of the server in Live environment
  • HTTP_SANDBOX: Connects with the endpoint of the server in Sandbox environment
  • TEST_OK: Emulates the connection where ok and the data received is correct
  • TEST_ERROR_IOEXCEPTION: Emulates an IOException that occurs during the communication
  • TEST_ERROR_HTTPEXCEPTION: Emulates a HttpException that occurs during the communication
  • TEST_UNAUTHORIZED: Emulates a 401 error - Unauthorized
