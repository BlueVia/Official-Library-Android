Detailed Description

Internal class to hold information about the makeCallSession method params data send to the gSDP It includes

  • firstCallParticipant: The address of the first participant involved in the call session. This parameter is mandatory.
  • secondCallParticipant: The address of the second participant involved in the call session. This parameter is mandatory.
  • firstParticipantDisplayedAddress: The incoming call identification that will be displayed on the first participant's phone. This parameter is optional.
  • secondParticipantDisplayedAddress: The incoming call identification that will be displayed on the second participant's phone. This parameter is optional.
  • mediaInfoList: This parameter identifies one or more media to be applied to the participants in the call sessions. This parameter is mandatory.
  • changeMediaNotAllowed: This flag specifies if the switching process across different media is not allowed.
