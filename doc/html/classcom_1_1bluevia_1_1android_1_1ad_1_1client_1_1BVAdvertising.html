Requests the retrieving of an advertisement. This functions can only be used in 2-legged-mode (OAuth token cannot have been included in the construction of the client)

Parameters:
adSpacethe adSpace of the Bluevia application (mandatory)
countrycountry where the target user is located. Must follow ISO-3166 (see http://www.iso.org/iso/country_codes.htm) (optional).
targetUserIdIdentifier of the Target User (optional).
adRequestIdan unique id for the request. (optional: if it is not set, the SDK will generate it automatically)
adPresentationthe value is a code that represents the ad format type (optional)
keywordsthe keywords the ads are related to (optional)
protectionPolicythe adult control policy. It will be safe, low, high. It should be checked with the application SLA in the gSDP (optional)
userAgentthe user agent of the client (optional)

the result returned by the server that contains the ad meta-data
