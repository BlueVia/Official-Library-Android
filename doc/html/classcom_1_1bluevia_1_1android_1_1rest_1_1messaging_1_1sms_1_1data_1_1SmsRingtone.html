Detailed Description

Class representing the SMSRingtone that will be send using the SMS Client API This type is composed of the following fields:

  • addressList: mandatory; max occurrences: minimum 1.
  • ringtone: mandatory; String
  • originAddress: mandatory; String

This implementation is not synchronized
