Detailed Description

This class implements a part of a Multipart post object that consists of a file. Two different types of files are supported:

  • text files, for the following content types:
    • text/plain
    • application/xml
    • application/json
  • binary files, for the following content types:
    • image/jpeg
    • image/bmp
    • image/gif
    • image/png
    • audio/midi
    • audio/wav
    • audio/mpeg
    • audio/amr
    • video/avi
    • video/x-ms-asf
    • video/mpeg
    • video/mpeg4
    • video/3gpp

Following the specifications of Bluevia API, the following transfer encodings will be used:

  • 8bit for text files, using UNICODE-8 charset.
  • base64 for binary files.
