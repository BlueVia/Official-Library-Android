Detailed Description

This class works as a filter to let developers notify the library about which fields of the UserTerminalInformation block they want to retrieve. Developers must set a true value in those fields they whish the library to retrieve from the gSDP and pass the object to the getUserTerminalInformation method of their DirectoryClient instance
