Detailed Description

Class representing the SMSLogo that will be send using the SMS Client API This type is composed of the following fields:

  • addressList: mandatory; max occurrences: minimum 1.
  • image: mandatory; byte[]
  • originAddress: mandatory; String

This implementation is not synchronized
